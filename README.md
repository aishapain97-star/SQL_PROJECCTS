# LAYOFF_DATA_CLEANING

Summary :
An end-to-end SQL project analyzing global tech layoffs from 2020-2023. This project showcases advanced data cleaning techniques and exploratory data analysis (EDA) using MySQL, transforming raw, messy data into actionable business insights regarding industry trends and company impacts.

Conducted an end-to-end Exploratory Data Analysis (EDA) on a global tech layoff dataset (2020-2023) using MySQL to uncover industry trends and company impacts.

Executed complex data cleaning protocols utilizing CTEs and Window Functions (ROW_NUMBER) to identify and remove duplicates across millions of data points.

Standardized raw data by normalizing categorical values, parsing strings to DATE formats, and strategically handling null values to ensure high data integrity for analysis.

Formulated advanced SQL queries utilizing DENSE_RANK and aggregate functions to calculate rolling totals and rank the highest-impacted companies over time.
